SOS Partner WordPress Package

Contenuto pacchetto:
- Plugin: sos-partner-bridge-lite
- Guida: partner-wordpress-guide.md

Installazione plugin:
1. Apri WordPress admin del sito partner.
2. Vai su Plugin > Aggiungi nuovo > Carica plugin.
3. Carica lo ZIP del plugin sos-partner-bridge-lite.
4. Clicca Installa ora.

Attivazione:
1. Vai su Plugin installati.
2. Attiva SOS Partner Bridge Lite.

Configurazione:
1. Apri menu SOS Partner Bridge.
2. Inserisci Partner ID, URL sito centrale e Shared secret.
3. Seleziona integration_mode.
4. Verifica i path endpoint richiesti dalla modalita.
5. Salva impostazioni.

Esecuzione test:
1. Esegui Test connessione.
2. Esegui Test handoff.
3. Esegui Test callback (se modalita payment_callback o combined).
4. Verifica gli esiti nella sezione Test e diagnostica.
